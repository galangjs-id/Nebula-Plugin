🌌 Nebula Project

Version: 1.0.0-Release
Author: @lanzzx1

A performance-focused plugin designed to improve device efficiency across various SoCs, with optimized system tweaks and gaming enhancements.

---

⚙️ Installation

Supported on:

- AxManager

Steps:

1. Download the latest release
2. Install via AxManager
3. Apply your preferred configuration

---

🚀 Features

🔹 Active Profiles

- Performance
- Balanced
- Battery Saver

🔹 System Configuration

- CPU Governor Tweak
- GPU Boost
- I/O Scheduler
- RAM Optimizer
- ZRAM Tweak

🎮 Game Manager

- Force 60 FPS
- Disable Background Apps
- High Priority Thread
- DND Mode

🌐 Network Optimization

- TCP Optimization
- UDP Buffer Boost
- WiFi Optimizer

🎯 Gaming Features

- Touch Sensitivity (Adjustable X & Y Axis)
- Touch Response (Latency & Polling Rate)
